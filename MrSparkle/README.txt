A demonstration of a simple text-drawing sparkle effect using CAEmitterLayer and CAShapeLayer.

Contents: 
 - a UILabel category that returns a path based approximately on a UILabel
 - A view controller that performs the animation

This demo project was used in a presentation at Ottawa CocoaHeads by Josh Svatek.


See: "Animating the drawing of a CGPath with CAShapeLayer", Ole Begemann
http://oleb.net/blog/2010/12/animating-drawing-of-cgpath-with-cashapelayer/

See: Fireworks Sample Project, Apple (spark and smoke PNGs are taken from this project)
